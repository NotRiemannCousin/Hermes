Detailed Description

template<SocketDataConcept SocketData = DefaultSocketData<>, class AcceptPolicy = DefaultAcceptPolicy<>, class TransferPolicy = DefaultTransferPolicy<>>
requires ServerSocketConcept<SocketData, AcceptPolicy, TransferPolicy>
struct Hermes::ListenerSocket< SocketData, AcceptPolicy, TransferPolicy >

Blocking listener socket. Owns the listening SocketFd handle and produces ServerSocket instances via AcceptOne() / AcceptAll().

The listening socket lives inside socketData.socket. AcceptPolicy drives both the bind/listen phase (Listen()) and the per-connection accept phase (AcceptOne()), including any protocol-level handshake (e.g. TLS ServerHandshake). TransferPolicy is forwarded as-is to every ServerSocket produced.
